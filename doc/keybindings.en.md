## Navigation and Files

| Keybinding | Description |
|------------|-------------|
| `<leader>ff` | Find files with the option to choose the split direction |
| `<leader>fg` | Live search across all project files |
| `<leader>fb` | Switch between buffers |
| `<leader>fh` | Search documentation |
| `<leader>t` | Telescope menu |
| `<leader>nn` | Create a new file |
| `<leader>fd` | Delete the current file with confirmation |

### Additional in file search (`<leader>ff`):
- `Enter` - open in the current window
- `h` - open in a horizontal split
- `v` - open in a vertical split  
- `l` - open on the left
- `j` - open on the bottom
- `n` - create a new file
- `d` - delete a file

## File Tree

| Keybinding | Description |
|------------|-------------|
| `<leader>e` | Focus the file tree |
| `<leader>E` | Toggle file tree visibility |
| `Enter` | Open file in the current window |
| `Alt+Enter` | Open in a horizontal split |
| `Ctrl+Enter` | Open in a vertical split |

## Code Navigation

| Keybinding | Description |
|------------|-------------|
| `<leader>sf` | Search in the current file |
| `<leader>sg` | Search the word under the cursor |
| `<leader>sr` | Find all references to a symbol |
| `<leader>sd` | Symbols in the current file |
| `<leader>sw` | Symbols in the workspace |
| `<leader>si` | Find implementations |
| `gd` | Go to definition |
| `gD` | Go to declaration |
| `gi` | Go to implementation |
| `gr` | Find all references |
| `K` | Show documentation |

## Window Management

| Keybinding | Description |
|------------|-------------|
| `Alt+←` | Move to the left window |
| `Alt+→` | Move to the right window |
| `Alt+↑` | Move to the upper window |
| `Alt+↓` | Move to the lower window |
| `<leader>ss` | Smart horizontal split |
| `<leader>vv` | Smart vertical split |
| `<leader>qc` | Close the current window |
| `<leader>ar` | Automatically equalize all windows |

## Working with Code

## Diagnostics and Error Navigation

| Keybinding | Description |
|------------|-------------|
| `[d` | Go to the previous error/warning |
| `]d` | Go to the next error/warning |
| `<leader>e` | Show diagnostics in a floating window |
| `<leader>q` | Add errors to the location list |

## Git Integration

| Keybinding | Description |
|------------|-------------|
| `<leader>gg` | Open Neogit |
| `]h` | Next git hunk |
| `[h` | Previous git hunk |

## Terminal

| Keybinding | Description |
|------------|-------------|
| `Ctrl+\` | Toggle terminal |

## Autocompletion

| Keybinding | Description |
|------------|-------------|
| `Ctrl+Space` | Show completion options |
| `Enter` | Select a completion option |
| `Tab` | Next option |
| `Shift+Tab` | Previous option |
| `Ctrl+e` | Cancel completion |
| `Ctrl+b` | Scroll documentation up |
| `Ctrl+f` | Scroll documentation down |

---

**Note:** `<leader>` is set to space by default.
