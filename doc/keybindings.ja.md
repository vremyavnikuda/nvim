# Neovim キーバインド

## ナビゲーションとファイル

| キー | 説明 |
|-----|------|
| `<leader>ff` | 分割方向を選んでファイル検索 |
| `<leader>fg` | プロジェクト全体をライブgrep |
| `<leader>fb` | バッファ切り替え |
| `<leader>fh` | ヘルプタグ検索 |
| `<leader>t` | Telescope メニューを開く |
| `<leader>nn` | 新規ファイル作成 |
| `<leader>fd` | 現在のファイルを削除（確認あり） |

### ファイル検索（`<leader>ff`）の追加操作

- `Enter` - 現在のウィンドウで開く
- `h` - 横分割で開く
- `v` - 縦分割で開く
- `l` - 左側で開く
- `j` - 下側で開く
- `n` - 新規ファイル作成
- `d` - ファイル削除

## ファイルツリー

| キー | 説明 |
|-----|------|
| `<leader>e` | ファイルツリーへフォーカス（buffer-local の LSP マップがない場合） |
| `<leader>E` | ファイルツリー表示切替 |
| `Enter` | 現在のウィンドウで開く |
| `Alt+Enter` | 横分割で開く |
| `Ctrl+Enter` | 縦分割で開く |

## コードナビゲーション

| キー | 説明 |
|-----|------|
| `<leader>sf` | 現在ファイル内検索 |
| `<leader>sg` | カーソル下の単語を検索 |
| `<leader>sr` | Telescope で参照を検索 |
| `<leader>sd` | Telescope でドキュメントシンボル |
| `<leader>sD` | Telescope で定義へ移動 |
| `<leader>sw` | Telescope でワークスペースシンボル |
| `<leader>si` | Telescope で実装を検索 |
| `<leader>ds` | ドキュメントシンボル（LSP） |
| `<leader>ws` | ワークスペースシンボル（LSP） |
| `gd` | 定義へ移動 |
| `gD` | 宣言へ移動 |
| `gi` | 実装へ移動 |
| `gr` | 参照一覧を表示 |
| `K` | ホバー情報を表示 |

## ウィンドウ操作

| キー | 説明 |
|-----|------|
| `Alt+Left` | 左ウィンドウへ移動 |
| `Alt+Right` | 右ウィンドウへ移動 |
| `Alt+Up` | 上ウィンドウへ移動 |
| `Alt+Down` | 下ウィンドウへ移動 |
| `<leader>ss` | スマート横分割 |
| `<leader>vv` | スマート縦分割 |
| `<leader>qc` | 現在ウィンドウを閉じる |
| `<leader>ar` | すべてのウィンドウを均等化 |

## コード操作

| キー | 説明 |
|-----|------|
| `<leader>fm` | 現在バッファをフォーマット |
| `<leader>rn` | カーソル下のシンボルをリネーム |
| `<leader>ca` | コードアクション |
| `<leader>ih` | インレイヒント切替 |
| `gcc` | 行コメント切替 |
| `gbc` | ブロックコメント切替 |

## 診断

| キー | 説明 |
|-----|------|
| `[d` | 前の診断へ移動 |
| `]d` | 次の診断へ移動 |
| `<leader>e` | 診断フロート表示（LSP バッファ内） |
| `<leader>q` | 診断を location list に追加 |

## Git

| キー | 説明 |
|-----|------|
| `<leader>gg` | Neogit を開く |
| `<leader>nw` | Nvim Wrapped を開く |
| `]h` | 次の git hunk |
| `[h` | 前の git hunk |

## ターミナル

| キー | 説明 |
|-----|------|
| `Ctrl+\\` | ターミナル切替 |

## 補完

| キー | 説明 |
|-----|------|
| `Ctrl+Space` | 補完を開始 |
| `Enter` | 補完候補を確定 |
| `Tab` | 次の候補 |
| `Shift+Tab` | 前の候補 |
| `Ctrl+e` | 補完を中止 |
| `Ctrl+b` | ドキュメントを上へスクロール |
| `Ctrl+f` | ドキュメントを下へスクロール |

---

**注記:** `<leader>` は `Space` に設定されています。
