## ナビゲーションとファイル

| キーバインド | 説明 |
|------------|------|
| `<leader>ff` | 開く方向を選択できるファイル検索 |
| `<leader>fg` | プロジェクト内をライブ検索 |
| `<leader>fb` | バッファの切り替え |
| `<leader>fh` | ドキュメント検索 |
| `<leader>t` | Telescope メニュー |
| `<leader>nn` | 新しいファイルを作成 |
| `<leader>fd` | 確認付きで現在のファイルを削除 |

### 🔍 ファイル検索の追加操作 (`<leader>ff`):
- `Enter` - 現在のウィンドウで開く
- `h` - 水平スプリットで開く
- `v` - 垂直スプリットで開く  
- `l` - 左に開く
- `j` - 下に開く
- `n` - 新しいファイルを作成
- `d` - ファイルを削除

## ウィンドウ管理

|------------|------|
| `Alt+←` | 左のウィンドウへ移動 |
| `Alt+→` | 右のウィンドウへ移動 |
| `Alt+↑` | 上のウィンドウへ移動 |
| `Alt+↓` | 下のウィンドウへ移動 |
| `<leader>ss` | スマート水平スプリット |

## 診断とエラー移動

| キーバインド | 説明 |
|------------|------|
| `[d` | 前のエラー/警告へ移動 |
| `]d` | 次のエラー/警告へ移動 |
| `<leader>e` | フローティングで診断を表示 |
| `<leader>q` | エラーをロケーションリストに追加 |

## 補完

| キーバインド | 説明 |
|------------|------|
| `Ctrl+Space` | 補完候補を表示 |
| `Enter` | 補完候補を選択 |
| `Tab` | 次の候補 |
| `Shift+Tab` | 前の候補 |
| `Ctrl+e` | 補完をキャンセル |
| `Ctrl+b` | ドキュメントを上へスクロール |
| `Ctrl+f` | ドキュメントを下へスクロール |
